# Assignment13_AI
# Generative AI Text Generation using GPT-style Model

## Overview
This project demonstrates a basic GPT-inspired text generation model trained on Project Gutenberg text using TensorFlow.

## Setup
- Python 3.9+
- TensorFlow
- Numpy, Pandas, Matplotlib

## How to Run
1. Open the notebook in Google Colab
2. Upload the dataset text file
3. Run all cells

## Output
- Training loss plots
- Generated text samples
